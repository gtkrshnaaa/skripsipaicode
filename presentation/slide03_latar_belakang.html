Latar Belakang

Perkembangan LLM telah mendorong lahirnya beragam asisten pemrograman, namun sebagian besar beroperasi sebagai ekstensi editor atau layanan cloud yang menimbulkan kekhawatiran terkait privasi dan kendali data.

CLI tetap penting bagi banyak pengembang karena sifatnya yang ringan, dapat diotomasi, dan mudah diintegrasikan dengan beragam alat.

Integrasi agen cerdas stateful dan proactive ke dalam CLI berpotensi mempercepat proses pengembangan perangkat lunak.

Paicode berjalan pada terminal lokal dan mengeksekusi tindakan langsung pada berkas proyek di workspace, namun mengirimkan cuplikan kode/konteks ke layanan LLM melalui API untuk inferensi.

Aspek Keamanan

Lokal

Path security, deny-list, pembatasan diff (max 500 baris)

API

Privasi bergantung pada kebijakan penyedia (Gemini)
