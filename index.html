Paicode

Workspace Controller • Diff-aware • CLI

Paicode adalah agen AI berbasis CLI yang menjalankan operasi berkas tingkat-aplikasi pada workspace proyek secara terkontrol. Terintegrasi LLM (Gemini), menerapkan path-security dan modifikasi berbasis diff.

Agentic CLI

Interaksi stateful via terminal dengan rencana aksi eksplisit: MKDIR, TOUCH, READ, WRITE, MODIFY, dst.

Workspace Controller

Operasi berkas tingkat-aplikasi pada ruang kerja proyek dengan path-security dan deny-list sensitif.

Diff-aware Modify

Perubahan terarah dengan patch/diff untuk meminimalkan risiko penimpaan tak diinginkan.
