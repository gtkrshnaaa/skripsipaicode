\chapter{Implementasi dan Hasil}
\label{chap:implementasi}

\section{Implementasi Paicode}
Uraikan proses instalasi (\texttt{poetry install}), konfigurasi API key (\texttt{poetry run pai config --set <KEY>}), dan cara menjalankan (\texttt{poetry run pai}).

\section{Contoh Sesi Interaktif}
Berikan contoh sesi terminal, output TREE/LIST, dan pembacaan/penulisan/\texttt{MODIFY} file.

\section{Evaluasi}
Jelaskan skenario uji, metrik evaluasi, dan bandingkan dengan proses manual atau alat lain.
